# document フォルダ

要求定義・設計・変更履歴などのドキュメントを格納します。

## ファイル一覧

| ファイル | 内容 |
|----------|------|
| **00_要求定義書_現状.md** | 現状のソースから書き起こした要求定義書。システム概要、ユーザ種別、機能一覧、データモデル、画面遷移、非機能要件、用語を記載。 |

## 今後の追加例

- `01_機能追加_〇〇.md` … 新機能の要求
- `02_変更履歴.md` … リリースごとの変更メモ
- `03_画面一覧.md` … 画面ID・URL・権限の一覧（00 の補足）
- `04_データ辞書.md` … テーブル・カラムの詳細（00 の補足）

必要に応じて 00_要求定義書_現状.md を更新し、大きな変更は別ファイルで追記していく運用を推奨します。
