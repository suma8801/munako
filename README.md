# このシステムについて

同窓会を開催するにあたって、出席の確認、連絡方法などを個別に記録することができる。
また、同級生の名前やクラスで検索することができる。

## ユーザ種別

ユーザは、一般ユーザとスタッフ、管理者に分ける、スタッフとは同窓会幹事であり
一般ユーザの参加・不参加・連絡不要・連絡日時、メモなどを入力できる。
管理者はスタッフを設定することができる。その際、メールアドレスと仮パスワードを設置する。


## ログイン機能

ユーザはログインID(メールアドレス)とパスワードによるログインを行う。
これらの情報はDBにあり、ログインパスワードを忘れた場合は、保存してあるメールアドレスにパスワード変更通知を送ることができる。

## 一般ユーザ向け表示

一般ユーザーでログインした場合は、検索ができるページが表示される。
この時のURLパスは、homes/regacy とする。
